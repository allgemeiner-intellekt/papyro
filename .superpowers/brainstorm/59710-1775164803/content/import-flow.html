What the user sees at each stage of the pipeline

Paper State Machine
Importing
PDF dropped. File copied to papers/{uuid}.pdf. Appears in paper list immediately.
Row shows: original filename, spinner icon, "Importing..." badge
↓ text extracted + identifiers parsed
Resolving
Identifiers found (or title extracted). Metadata fetch in progress.
Row shows: extracted title (if any) or filename, "Resolving..." badge
↓ metadata found
↓ fetch failed / no identifiers
Resolved
Full metadata populated. PDF renamed. Index written.
Row shows: title, authors, year, journal — normal paper row with reading status badge
Unresolved
No metadata. PDF keeps temp name. User can edit manually or retry.
Row shows: filename, "Unresolved" badge, edit button
